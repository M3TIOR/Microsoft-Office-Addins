Support

If you have any issues with the app that this page could not answer, please look for extra help on this project's Github issues page. If you cannot resolve your issue that way please create a new issue on Github that explains how you originally recieved the error, what Operating System you're running on, what browser you're using as well as the version, and if you can reproduce the error. If you can, please be sure to also include some screenshots or debug logs of the problem. A developer will be with you within two to three days. If it takes longer to process your issue and resolve it, please be patient.
